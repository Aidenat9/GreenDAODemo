# GreenDAODemo
greenDAO demo. simple use ;one to many...
####注意：
```
  greendao {
        //版本号，升级时可配置
        schemaVersion 3
        daoPackage 'com.tmgg.greendao.gen'//这个是生成代码保存的包名
        targetGenDir 'E:\\GreenDAODemo\\app\\src\\main\\java'//保存到java代码路径
    }
```
- 升级时候别忘了改版本号，targetGenDir地址一定要写对
